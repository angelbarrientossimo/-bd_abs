El objetivo de esta función es encontrar todos los POIs dentro de un radio específico desde la ubicación del usuario.

```python
def find_by_radius(lat, lon, distance=2000):
    poi_ids = r.geosearch(
        "poi:locations",
        longitude = lon,
        latitude = lat,
        radius=distance,
        unit = "m"
    )
    for poi_id in poi_ids:
        name = r.hget("poi:info",poi_id)
        print(f"{name}({poi_id})")
```

Tarea 3 (Opcional): Búsqueda del “más cercano”
El objetivo será encontrar el punto de interés único más cercano al usuario, sin importar la distancia.

Debes de:

Crear un script find_nearest.py(lat, lon) que recogerá la ubicación del usuario.
Usa el comando GEOSERACH para encontrar el POI más cercano (solo 1).
Asegúrate de pedir también la distancia.
Imprime el resultado: El POI más cercano es "Nombre del POI", que está a X.XX km.


```python
def find_nearest(lat,lon,distance):
    poid_busqueda = r.geosearch("poi:locations",longitude=lon,latitude=lat,radius=distance,unit="km")
    id = poid_busqueda[0]
    nombre_ciudad = r.hget("poi:info", id)
    #Añadir el valor que pone el usuario, despues geodist, despues borrar el usuario ese creado (para poder reutilizarlo)
    r.geoadd("poi:locations", (lon, lat, "Usuario:"))
    distanciaUsuarioCiudad = r.geodist("poi:locations","Usuario:","poi_019",unit="km")
    print(f"La ubicación más cercana es: {poid_busqueda[0]} Nombre = {nombre_ciudad} que está a {distanciaUsuarioCiudad} km")
    r.zrem("poi:locations","Usuario:")
    
```
