4- Pirámide de asteriscos

# Solicitar número impar
while True:
    n = int(input("Introduce un número impar: "))
    if n % 2 != 0:
        break
    print("El número debe ser impar. Inténtalo de nuevo.")

# Dibujar pirámide con base = n
altura = (n + 1) // 2
for i in range(1, altura + 1):
    asteriscos = 2 * i - 1
    espacios = (n - asteriscos) // 2
    print(' ' * espacios + '*' * asteriscos)

20. Ordenar lista de números

entrada = input("Ingresa una lista de números separados por comas: ")
